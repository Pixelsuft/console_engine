# Console Engine
Python Console Engine for windows
# Functions
1) Get Async Key State<br />
2) Wait for input<br />
3) Print with text align<br />
4) Play wav sounds<br />
5) FPS Clock tick<br />
6) Clear canvas<br />
7) Fill canvas<br />
8) Draw Shape<br />
9) Fill Shape<br />
10) Change Title
# Install
```
python -m pip install console-engine
```
